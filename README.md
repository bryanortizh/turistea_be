PROYECTO TURISTEA PASOS PARA INSTALAR Y LEVANTAR EL PROYECTO

1. Ejecutar comando npm install (version de node v22.16.0)
2. Ejecutar npm run build contruira el proyecto
3. Ejecutar comando npm run dev este ejecutara un archivo nodemon que a cada actualizacion volvera a buildear el proyecto.
4. Script de base de datos adjuntada en el envio del proyecto
5. Importar la coleccion de postman
